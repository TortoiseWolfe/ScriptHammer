# Forking Feedback

This file is a placeholder for documenting feedback from projects that fork ScriptHammer.

If you fork this template and encounter issues or have suggestions for improvement, please document them here and consider submitting a pull request to help future users.

## How to Contribute Feedback

1. Document the issue you encountered
2. Describe any workarounds or solutions you found
3. Suggest improvements to the template
4. Submit a PR or open an issue at [ScriptHammer Issues](https://github.com/TortoiseWolfe/ScriptHammer/issues)
